Buttons

Buttons will follow sizing guidelines based on their perceived priority in the app.

General Guidelines

  • Text colour is black (#000)
  • Border radius is 10px

Buttons Without Text

Widths and Height
  • High Priority: 80px
  • Medium Priority: 60px
  • Low Priority: 45px

Buttons With Text

Height

Height is the same size as described for buttons without text

Width

Buttons with text are 3x the button height

With Text

High Priority

3X

X = 80px

Medium Priority

3X

X = 60px

Low Priority

3X

X = 45px

Margins

Below are the total margin widths between elements. Therefore, each element will have half of the listed margin sizes of the value stated below.

  • High priority: 12px
  • Medium priority: 24px
  • low priority: 40px

Colours

Drop Shadow

  • 4x spread in X and Y with 10px blur in the colour Charcoal (#373F47) to indicate clickability.

Colours

  • High priority: Pumpkin (#FF8C42) or Silver Lake Blue (#6C91C2) to accent the highest level of importance
  • Medium priority: Silver Lake Blue (#6C91C2)
  • Low priority: Silver Lake Blue (#6C91C2) or Charcoal (#373F47)

Post Buttons

Post Buttons deviate slightly from the general button guidelines as it is one of the main features of the app. These buttons are high-priority and therefore 80px high and are fill the width of the screen. In order to preserve the use of Pumpkin colour as a priority accent, the colour used for post buttons is Silver Lake Blue (#6C91C2) because the presence of many post buttons may negate the importance of the Pumpkin colour.

Padding is 1em for the vertical (height) axis and 0.5em for the horizontal (width) axis.

Quiz Option Buttons

Quiz Option Buttons should follow the guidelines established for Post Buttons. Default state uses a fill colour of Silver Lake Blue (#6C91C2), Selected state Charcoal (#373F47) with a text colour of Beige (#FDFAE3). Feedback state uses a fill colour of Pumpkin (#FF8C42) with a 3px solid stroke in Charcoal colour.

Navigation Bar

Nav bar buttons follow the medium priority button guidelines.

  • Each icon is 60x60px size
  • Margin between each icon and between icons is 12px per icon side (24px total margin)

In the default state, the icons are outlined in Charcoal (#373F47) with a Beige (#FDFAE3) fill. In the active/clicked state, the icons will change to a Charcoal fill.
